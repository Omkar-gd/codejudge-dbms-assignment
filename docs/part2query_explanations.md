# Part 2 - SQL Query Explanations

## Query 1
Lists all active students with student details.

## Query 2
Finds students whose email is missing or invalid.

## Query 3
Displays problems with Easy or Medium difficulty.

## Query 4
Shows the latest 20 submissions.

## Query 5
Counts total submissions made by each student.

## Query 6
Calculates average score for each problem.

## Query 7
Finds students who made more than 5 submissions.

## Query 8
Displays contests and total problems in each contest.

## Query 9
Finds the highest score achieved for each problem.

## Query 10
